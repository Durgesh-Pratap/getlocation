# 📍 GeoPulse Core – Smart Location Tracking & Geo Alert System

🚀 **Live Demo:**
👉 [https://durgesh-pratap.github.io/getlocation/](https://durgesh-pratap.github.io/getlocation/)

---

## 🧠 Problem Statement

Traditional alerts are **time-based**, but real-world travel (especially trains) is **dynamic and unpredictable**.

* Delays are common
* Arrival times change
* Users miss stops due to sleep or uncertainty

👉 Result: **Missed destinations**

---

## 💡 Solution

**GeoPulse Core** shifts from *time-based alerts* → **location-based intelligence**

Instead of asking:

> “When will I reach?”

We solve:

> “Notify me when I am near my destination”

---

## ⚙️ Core Features

### 📡 Live GPS Telemetry

* Real-time Latitude & Longitude
* Altitude (if available)
* Accuracy tracking
* Continuous GPS feed

---

### 🎯 Smart Navigation Target (Pin System)

* Tap map to set destination
* Visual marker placement
* Extendable to geo-fencing alerts

---

### 📍 Location Capture & Annotation

* Save current location with remarks
* Timestamped entries
* Use cases:

  * Site inspection
  * Travel logs
  * Field operations

---

### 🔗 Share Location

* One-click Google Maps link
* Copy to clipboard
* Easy sharing across apps

---

### 📊 Location History

* Persistent local storage
* Structured logs with:

  * Time
  * Coordinates
  * Remarks

---

### 📤 Export Data

* Export as CSV
* Ready for:

  * Reporting
  * Analytics
  * Audit trails

---

### 📏 Map Tools

* 📏 Distance measurement (Ruler Mode)
* 🎯 Target marking
* Interactive map controls

---

### 🌙 Night Mode

* Dark UI for low-light usage
* Improved usability during travel

---

## 🏗️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Maps:** Leaflet.js (OpenStreetMap)
* **Storage:** LocalStorage
* **Geolocation:** Browser Geolocation API

---

## 📱 Usage

### 1. Start Tracking

Click **“Start GPS Feed”** to begin live tracking

### 2. Save Location

* Add remark
* Click **Save Point**

### 3. Set Target

* Click **Nav Target**
* Tap on map

### 4. Export Data

* Click **Export CSV**

---

## ⚠️ Limitations

* Browser-based GPS:

  * May not provide altitude consistently
  * Accuracy varies (±10–100m)
* Background tracking not reliable in browser

👉 For production:

* Android app required (Capacitor / Native)
